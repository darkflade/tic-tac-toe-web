# Крестики-нолики (Веб-версия)

Веб-версия классической игры «Крестики-нолики» с искусственным интеллектом, написанная на чистом JavaScript (ES6+), HTML и CSS.

- **Адаптивный ИИ**: Сложность ИИ подстраивается под размер поля.
- **Сохранение истории**: Все партии сохраняются в локальной базе данных вашего браузера (IndexedDB), и вы можете просмотреть историю игр.
- **Настраиваемое поле**: Вы можете выбрать размер игрового поля от 3x3 до 10x10.
- **Без бэкенда**: Игра полностью работает на стороне клиента и может быть размещена на любом статическом хостинге.

---

## Как играть

1.  **Откройте игру в браузере.**
    -   Игра доступна по ссылке: **[ЗДЕСЬ БУДЕТ ВАША ССЫЛКА GITHUB PAGES]**

2.  **Настройте партию.**
    -   Введите свое имя.
    -   Выберите размер игрового поля (от 3 до 10).
    -   Нажмите "Начать новую игру".

3.  **Игровой процесс.**
    -   Игроку и компьютеру случайным образом присваиваются "X" или "O". Первым всегда ходит "X".
    -   Кликайте на пустые ячейки, чтобы сделать ход.
    -   Цель игры — собрать линию из своих фигур. Длина выигрышной линии:
        -   **3** для полей размером 3x3.
        -   **5** для полей размером от 4x4 и больше.

---

## Технологии

-   **JavaScript (ES6+)**: Вся логика игры, включая ИИ.
-   **HTML5 / CSS3**: Интерфейс и стилизация.
-   **IndexedDB**: Клиентская NoSQL база данных для хранения истории игр.
-   **GitHub Pages**: Хостинг для статического сайта.
